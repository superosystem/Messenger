# MERN SOCIAL MEDIA - Stories

## Features

* User profile,
* User following each other,
* Who follow suggestions,
* Newsfeed with posts from followed users,
* Listing posts by user,
* Like posts,
* Commenting on posts.

Users: This contains all user-related stories.
