# Reddit Spring - [Backend]

Reddit website use Spring Boot.

## Run Locally

Use the package manager [maven](https://maven.apache.org/) to install your computer.

```bash
# Build Application
mvn clean build

# Running Application
java -jar /build/Backend-Application.jar
```
